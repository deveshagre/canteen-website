# Grain & Gosips - Campus Cafe Menu

A modern, responsive food ordering menu website for a campus cafe. Built with vanilla JavaScript, HTML, and CSS.

## Features

- 🍽️ **Menu Categories**: Veg Rice & Noodles, Non-Veg Rice & Noodles, Burgers, Pizza, Sandwiches, Snacks, and Beverages
- 🛒 **Shopping Cart**: Add items to cart with quantity controls
- 🏷️ **Veg/Non-Veg Badges**: Clear labeling for dietary preferences
- 📱 **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- 🎨 **Modern UI**: Clean, elegant design with smooth animations

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Or serve it using a local server:
   - **Python**: `python -m http.server 8000`
   - **Node.js**: `npx http-server`
   - **VS Code**: Use the Live Server extension

## Usage

- Browse menu items by category
- Click "Add to Cart" to add items
- View cart by clicking the cart icon in the header
- Adjust quantities using + and - buttons
- Minimum order value: ₹350

## Technologies Used

- HTML5
- CSS3 (with CSS Variables, Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- Google Fonts (Lato, Playfair Display)
